# Installation

Install it:
```zsh

curl -fSL  "https://raw.githubusercontent.com/purplehippo911/Jojos-proj/main/madeinheaven.sh" | sh

```

Add to path:
```sh
export PATH=$PATH:"/usr/local/jojos/bin"
```

Run:
```sh
jojos josu
```

Make sure to turn on audio. the song will play.

To uninstall:
```sh
zahando.sh
```
